# Application Web – Quiz interactif style Microsoft Forms

## Description du projet

Ce projet est une application web de quiz développée en HTML, CSS et JavaScript pur (vanilla).  
L’objectif principal était de créer un quiz interactif et de découvrir les bases du développement web.

Bien que le code soit encore mal structuré et peu optimisé (code spaghetti typique de mes débuts), le projet démontre ma capacité à créer une application fonctionnelle côté frontend, à gérer l’interaction utilisateur et à mettre en place une logique simple pour le flow du quiz.

## Fonctionnalités principales

### Mode Administrateur
- Modifier le quiz existant : ajouter, éditer ou supprimer des questions et réponses
- Gérer le contenu affiché aux utilisateurs

### Mode Utilisateur
- Réaliser le quiz question par question
- Validation des réponses et navigation entre les questions
- Interface simple et interactive
- Feedback visuel pour les réponses
- Suivi de la progression du quiz

### Logiciel entièrement frontend
- Pas de backend : tout est stocké dans le code et le navigateur

## Technologies utilisées

- HTML / CSS pour la structure et le style
- JavaScript (vanilla) pour la logique et l’interaction utilisateur
- Programmation orientée fonctions et manipulation du DOM
